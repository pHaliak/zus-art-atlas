# ZUŠ Art Atlas – pravidlá projektu

## Zlaté pravidlo
Každá funkcia musí byť navrhnutá tak, aby ju učiteľ vedel použiť intuitívne, bez návodu a s minimálnym počtom kliknutí.

## Základné pravidlá
1. Rozhranie musí byť jednoduché.
2. Komplexnosť patrí dovnútra systému, nie na obrazovku.
3. Osnovy výtvarného odboru sú hlavný odborný rámec.
4. Téma je nosič vzdelávacieho cieľa, nie hlavný organizačný princíp.
5. Obrazové prílohy sú rovnako dôležité ako text.
6. Ku každej príprave patria vhodní umelci a významné diela.
7. Galéria žiackych prác sa zobrazuje ako koláž bez mien žiakov.
8. Výsledok vyhľadávania nikdy nesmie zostať prázdny.
